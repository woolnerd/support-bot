Support Chat Widget Demo

Features

  • Lightweight and embeddable
  • Customizable themes and colors
  • Real-time messaging
  • Responsive design
  • Persistent chat history
  • Easy configuration via data attributes

Configuration

The widget is configured with the following options:

<script src="dist/widget.js" data-endpoint="https://your-n8n-instance/webhook/support-chat" data-theme="light" data-primary-color="#007bff" data-title="Support Chat" data-welcome-message="Hi! How can we help you?" data-position="bottom-right" data-auto-open="false"></script>

n8n Integration

This widget communicates with an n8n workflow backend. The widget sends messages in this format:

{ "message": "Hello, I need help", "userId": "user_123", "sessionId": "session_456", "timestamp": "2024-01-15T10:30:00Z", "userInfo": { "name": "John Doe", "email": "john@example.com" } }

And expects responses in this format:

{ "response": "Hi! How can I help you today?", "messageId": "msg_789", "timestamp": "2024-01-15T10:30:05Z", "type": "bot" }

Try It Out

Click the chat button in the bottom-right corner to test the widget. Since there's no backend configured, you'll see an error message, but you can see the UI in action.
